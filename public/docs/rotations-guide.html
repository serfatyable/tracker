Rotation Management Quick Guide

This lightweight page exists so the in-app “Open guide” link always resolves. Replace the URL shown in the app by setting NEXT_PUBLIC_ROTATIONS_GUIDE_URL to your preferred documentation site if you have a richer resource.

Recommended workflow
  1. Create a new rotation from the admin dashboard.
  2. Assign tutors using the “Manage owners” action.
  3. Import or build the curriculum tree (CSV/XLSX templates are available in the app).
  4. Run the maintenance tool after large imports to deduplicate nodes.

Useful resources

  • Curriculum Excel template
  • Curriculum CSV template
  • ROTATION_TEMPLATE_GUIDE.md (full reference)

Need a richer handbook? Host it anywhere (Notion, Confluence, etc.) and set NEXT_PUBLIC_ROTATIONS_GUIDE_URL so admins are sent there instead.
